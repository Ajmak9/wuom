# Example — Olivar (Modular)

This example shows how WUOM can be applied to a real environment: an olive grove.

## Situation

Managing an olive grove with multiple variables:
- soil condition 
- tree health 
- biodiversity 
- seasonal cycles 

## Geometry

Modular (3x3)

## Structure

The system is organized into three layers:

### Input
- Soil condition 
- Climate 
- External factors 

### Process
- Tree growth 
- Biological activity 
- Human intervention 

### Output
- Olive production 
- Oil quality 
- Ecosystem balance 

## Result

The olive grove is no longer seen as isolated elements, 
but as a structured system.

This allows:
- better observation 
- clearer decisions 
- reduced unnecessary intervention 

## Notes

WUOM does not simplify reality. 
It structures it.

The system becomes visible without losing complexity.
