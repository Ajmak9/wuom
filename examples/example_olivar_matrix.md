# Example — Olivar (Matrix)

This example shows how WUOM can be applied using the Matrix geometry (8x8) in an olive grove.

## Situation

An olive grove is not a simple system.

It includes multiple interconnected layers:
- soil 
- trees 
- biodiversity 
- climate 
- time 

## Geometry

Matrix (8x8)

## Structure

Instead of a simple input-process-output model, 
the system is expanded into a network of relationships.

### Axes

**Horizontal (Space / Elements)**
- Soil 
- Roots 
- Tree 
- Canopy 
- Biodiversity 
- Microfauna 
- Climate interaction 
- Human presence 

**Vertical (Time / Processes)**
- Initial condition 
- Growth 
- Interaction 
- Stress 
- Adaptation 
- Recovery 
- Maturity 
- Decline 

## Result

Each point in the matrix represents a state of the system.

This allows:
- detecting patterns 
- understanding interactions 
- observing changes over time 

The olive grove is no longer seen as parts, 
but as a living network.

## Notes

Matrix does not simplify reality. 
It reveals its structure.

WUOM becomes more powerful as complexity increases.
