# WUOM Documentation

This folder contains the theoretical and explanatory documents of WUOM.

## Files

- MODULAR.pdf → introduction to simple structures (3x3)
- MATRIX.pdf → complex systems (8x8)
- SPHERICA.pdf → multidimensional relationships
- EXTENDED.pdf → flow and coexistence across geometries

## Purpose

These documents explain the structure and logic behind WUOM.

They are not instructions, but frameworks to understand and apply the system.

## Notes

WUOM is not a closed methodology.

It is a structural model designed to work with complexity.
